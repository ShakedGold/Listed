(static) Rank(term, option) → {number}

Ranks the term in the option by counting the number of letters in the term that are in the option not depending on the case.
Parameters:
Name Type Description
term string The partial term to search for (e.g. "abc").
option string The option to search through (e.g. "AbcDef").

(static) Search(term, options, dontUseFilter) → {Array}

Searches through the options for the term and returns a filtered and sorted list of options using Rank.
Parameters:
Name Type Description
term string The partial term to search for (e.g. "abc").
options Array The options to search through (e.g. ["abcdef", "def", "ghi"]).
dontUseFilter boolean If true, the filter will not be used, and all options will be returned.
Author:
  • ShakedGold
Source:
  • Algorithm.js, line 15
Returns:
The options sorted by rank (e.g. ["abcdef", "ghi", "def"]).
Type
Array
Example
Search("de", ["def", "abcdef", "ghi"]) -> ["def", "abcdef"];
